# DAC-Tetraperoxometalates SI ZENODO\_BADGE\_HERE
This repository contains the computational supporting information for the manuscript 'Electronic Structure and CO2 Reactivity of Group IV/V/VI Tetraperoxometalates' submitted to the *Journal of Physical Chemistry A*. All geometry optimizations were carried out in Gaussian 16 Rev A.03<sup>1</sup>. ESIgen<sup>2</sup>.
